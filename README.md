# School Attendance & Student Exit Tracking System

Complete, scalable solution using Google Sheets, Google Apps Script, and a professional classic website frontend.

## Quick Start

1. **Create Google Sheet** - Follow structure in `docs/sheet-structure.md`
2. **Deploy Apps Script** - Copy code from `backend/` folder
3. **Configure Frontend** - Update API URL in `frontend/js/config.js`
4. **Launch** - Open `frontend/index.html`

## Features

- ✅ Bulk attendance marking by class
- ✅ Student exit/pick-up logging
- ✅ Real-time reports and analytics
- ✅ Mobile responsive design
- ✅ Scalable to 100,000+ students
- ✅ Append-only audit trail

## Project Structure

```
/backend          - Google Apps Script code
/frontend         - HTML/CSS/JS website
/docs             - Setup guides
```
